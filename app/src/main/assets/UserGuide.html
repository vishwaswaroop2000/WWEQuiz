WWE Quiz Guide
This is the guide to the WWE Quiz App. On starting you would get the following screen
You can click on an available user or add a new user by clicking the 'Add a new User' Upon which a dialogue box would open
You can also choose to listen to notifcations about the latest questions and a foresground service would be running to keep you notified.
After choosing a user to start the game. Your Home can be changed by clicking on different icons and that will icon will be your home page for that user. This is done using SharedPreferences. You can also check the users statistics and also opt to change the profile at this time.
After choosing to Start the game you can choose the Levels, and then start the game. (Note: Connection to internet will be checked such that you will not be able to start the game if theres no internet connection)
Once the game is started, you will be given fifteen questions in each level. In Level easy, you would be given text-based questions. In Medium level, you would be shown images of wrestlers, as shown in the picture. And in the hard level, you would be played audio clips of the wrestler.
After the game is over you would be the Feedback to your quiz where you will be made known of your current scores, and if any level is reached.
